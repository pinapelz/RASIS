# RASIS
You can follow RASIS at [@rasis@sakurajima.social
](https://sakurajima.social/@rasis)

A simple Fediverse bot that relays information about various arcade games. Pulls from data generated by [573-updates](https://github.com/pinapelz/573-updates)
- Configurable rate limit, posting only X posts every hour to prevent spamming feed

Posted data get recorded to `rasis.db` by default (don't worry its a very simple schema)
